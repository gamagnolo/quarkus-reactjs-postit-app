# The Cloud Ready "Post-it" application
A Cloud Ready "Post-it" app composed of

- a Quarkus backend,
- a Nodejs/Reactjs frontend
- a Mongodb database

and deployed on OpenShift 4.3.1 cluster through CodeReady Workspaces 2.0
